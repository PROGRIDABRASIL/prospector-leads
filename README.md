# Prospector de Leads - Progrida Brasil

App de prospecção de leads usando Gemini AI + Google Search.

## Como rodar localmente
```
npm install
npm run dev
```

## Deploy no Vercel
1. Suba este código pro GitHub
2. Conecte o repositório no vercel.com
3. Deploy automático (sem configuração extra necessária)

## Uso
1. Acesse aistudio.google.com/app/apikey e gere uma API Key gratuita do Gemini
2. Cole a chave no campo do app
3. Escolha o segmento e a cidade
4. Clique em Buscar Leads
